## 📋 Resumen del Proyecto

Drakon Store Web es un sistema de e-commerce completo desarrollado siguiendo metodologías de ingeniería de software. Este proyecto incluye desde la elicitación de requisitos hasta el prototipado de interfaces, siguiendo las mejores prácticas de desarrollo de software.

## 📁 Estructura del Proyecto

Este proyecto de ingeniería de software sigue una estructura organizada por **fases de desarrollo**, facilitando la navegación y comprensión del flujo de trabajo.

## 🎯 Metodología Aplicada

Este proyecto sigue la metodología de **Durán & Bernárdez (2002)** para la ingeniería de requisitos y desarrollo de software, implementando:

- ✅ **Elicitación sistemática** de requisitos
- ✅ **Modelado UML** completo del sistema
- ✅ **Prototipado iterativo** de interfaces
- ✅ **Validación** con stakeholders
- ✅ **Documentación estructurada** por fases

## 🏆 Logros de la Reorganización

### ❌ **Problemas Anteriores:**
- Archivos dispersos sin organización lógica
- Referencias confusas como "Corte 3"
- Rutas de imágenes incorrectas
- Nomenclatura inconsistente
- Documentos sin estructura profesional

### ✅ **Soluciones Implementadas:**
- Estructura organizada por fases de ingeniería de software
- Nomenclatura consistente y descriptiva
- Todas las rutas de imágenes corregidas
- Formato visual uniforme en todos los documentos
- Documentación completa con tablas y estadísticas
